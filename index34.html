Google Drive (Overflow Storage)
Gunakan Service Account dari Google Cloud Console. File akan otomatis disimpan ke Drive saat R2 penuh, dan tetap bisa dibuka dari website.
Cara dapat refresh_token:
1. Buka Google Cloud Console → OAuth 2.0 Client IDs → buat credentials type Web application
2. Tambah https://developers.google.com/oauthplayground ke Authorized redirect URIs
3. Buka OAuth Playground → klik ⚙️ → centang "Use your own OAuth credentials" → masukkan client_id & client_secret
4. Pilih scope https://www.googleapis.com/auth/drive → Authorize → Exchange → copy refresh_token
Limit belum diset — upload ke Drive diblokir hingga limit diisi dan disimpan.

Panduan Recovery
Jika file R2 terhapus:
  1. Download Manifest Dokumen sekarang (sebelum terjadi masalah)
  2. Buka Cloudflare R2 Dashboard → bucket kamu
  3. Cocokkan kolom r2_key di CSV dengan file yang masih ada
  4. File yang hilang tandai di kolom status_recovery
  5. Minta ulang file dari pegawai yang bersangkutan
Jika konfigurasi R2 salah:
  1. Jangan hapus data di tabel dokumen — r2_key masih valid
  2. Perbaiki config di halaman Pengaturan → Cloudflare R2
  3. Test dengan upload 1 file kecil setelah perbaikan
