Panduan Recovery
Jika file R2 terhapus:
  1. Download Manifest Dokumen sekarang (sebelum terjadi masalah)
  2. Buka Cloudflare R2 Dashboard → bucket kamu
  3. Cocokkan kolom r2_key di CSV dengan file yang masih ada
  4. File yang hilang tandai di kolom status_recovery
  5. Minta ulang file dari pegawai yang bersangkutan
Jika konfigurasi R2 salah:
  1. Jangan hapus data di tabel dokumen — r2_key masih valid
  2. Perbaiki config di halaman Pengaturan → Cloudflare R2
  3. Test dengan upload 1 file kecil setelah perbaikan
